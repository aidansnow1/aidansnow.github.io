Langage Mot clé Utilité
HTML html L'élément racine de la page
Doctype Qui définit le type de document
head Qui contient les métadonnées
body Qui englobe le contenu principal ; et les balises de titre
h1 à h6 Qui structurent le contenu et indiquent sa hiérarchie
CSS selector Un sélecteur est utilisé pour "sélectionner" un ou plusieurs éléments HTML auxquels un style doit être appliqué
property Une propriété CSS est une caractéristique d'un élément que vous voulez modifier, comme la couleur ou la police
value C'est le réglage que vous donnez à une propriété CSS
declaration Une déclaration est une seule règle CSS constituée d'une propriété et d'une valeur, comme color red
important Cette déclaration permet de forcer l'application d'une règle de style, l'emportant sur d'autres règles conflictuelles
Javascript function Ce mot-clé est utilisé pour définir une fonction, qui est un bloc de code réutilisable et structuré pour accomplir une tâche spécifique
let permet de déclarer des variables dont la portée est limitée au bloc (entre {}) dans lequel elles sont déclarées
if Ce mot-clé est utilisé pour créer des structures conditionnelles, permettant d'exécuter un bloc de code uniquement si une certaine condition est vraie
for Le mot-clé for est employé pour créer une boucle, c'est-à-dire répéter un bloc de code un certain nombre de fois
return Ce mot-clé est utilisé à l'intérieur d'une fonction pour spécifier la valeur que la fonction doit renvoyer une fois son exécution terminée
